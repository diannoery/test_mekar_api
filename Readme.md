**API ROUTES**
- **User**
1. /user [POST] -> tambah data
2. /user [GET] -> mengambil semua data
3. /user/{id} [GET] mengambil data sesuai id
4. /user [PUT] -> mengedit data sesuai id
5. /user/{id} [DELETE] -> delete data
